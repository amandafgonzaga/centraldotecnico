Descrição do Projeto
A Central do Técnico é uma plataforma online dedicada a cursos de manutenção e reparo de dispositivos eletrônicos, como celulares, tablets e computadores. O site proporciona uma navegação simples e direta, com informações detalhadas sobre os cursos oferecidos, depoimentos de alunos e um formulário de contato.

Funcionalidades
Navegação fácil pelas seções do site, incluindo:
Cursos disponíveis
Depoimentos de alunos
Informações de contato
Seção de galeria com imagens dos cursos e atividades.
Formulário de contato para dúvidas e informações adicionais.
Links para redes sociais para interação com a comunidade.
Tecnologias Utilizadas
HTML: Estrutura básica do site.
CSS: Estilização do layout e design responsivo.
JavaScript: Interatividade, incluindo a expansão de detalhes dos cursos.
Google Maps: Para a visualização da localização física da Central do Técnico.
